A alias wrapper, it's easy to make linux alias and move between folders.

# Installation

    source wgo.sh

# Usage:

* set tag

        $ cd /foo/bar
        $ wgo foobar
        $ cd /foo/bar/2
        $ wgo foobar2

* goto folder

        $ foobar
        $ pwd
        $ /foo/bar
        $ foobar2
        $ pwd
        $ /foo/bar/2

* list tags

        $ wgo
